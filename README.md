# Sistema-rescate-GPS
Proyecto de simulación e interfaz de rescate con GPS
# 🚑 Sistema de Visualización y Análisis de Datos GPS para Operaciones de Rescate

## 📌 Descripción

Este proyecto presenta el desarrollo de una herramienta para la visualización, monitoreo y análisis de datos GPS transmitidos mediante tecnología LoRaWAN, orientada a operaciones de rescate en zonas rurales o de difícil acceso.

## 🎯 Objetivo

Desarrollar una herramienta devisualización y análisis de datos provenientes de GPS a través de LoRaWAN para la ubicación de las víctimas en operaciones de rescate.

## ⚙️ Funcionalidades

* 📍 Visualización de trayectoria en mapas interactivos
* 📊 Análisis de distancia y velocidad
* 🧪 Simulación de datos GPS
* 🖥️ Interfaz gráfica para monitoreo

## 🗂️ Estructura del proyecto

* `Simulación.ipynb`: análisis y procesamiento de datos
* `Interfaz_Rescate.py`: interfaz principal del sistema
* `mapa_tracker.html`: visualización geográfica
* `*.png`: resultados gráficos
* `*.csv / *.xlsx`: datos de entrada
* `Informe_GPS.pdf`: documentación del sistema

## 🛠️ Tecnologías utilizadas

* Python
* Jupyter Notebook
* Anaconda
* Librerías: pandas, matplotlib, folium

## ▶️ Cómo ejecutar

1. Abrir el notebook `Simulación.ipynb` en Jupyter Notebook
2. Ejecutar las celdas paso a paso
3. Ejecutar `Interfaz_Rescate.py` para visualizar la interfaz
